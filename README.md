# Web-Cars

Aplicação web para compra e venda de carros desenvolvida com React e TypeScript, onde usuários autenticados podem publicar, visualizar e gerenciar anúncios de veículos.

## 📌 Sobre o projeto

O **Web-Cars** é uma plataforma web que permite que usuários anunciem seus carros para venda de forma simples e organizada.

A aplicação possui autenticação de usuários, dashboard individual para gerenciamento dos anúncios e upload de imagens em nuvem, proporcionando uma experiência semelhante a plataformas reais de marketplace automotivo.

---

## ✨ Funcionalidades

- Cadastro e login de usuários com Firebase Authentication  
- Listagem de todos os carros anunciados na Home  
- Página de detalhes do carro (`/car/:id`)  
- Dashboard individual com anúncios do usuário  
- Cadastro de novos anúncios de carros  
- Exclusão de anúncios publicados  
- Upload e armazenamento de imagens com Cloudinary  
- Validação de formulários com React Hook Form  
- Navegação SPA com React Router  

---

## 🚀 Tecnologias utilizadas

Este projeto foi desenvolvido utilizando as seguintes tecnologias:

- React
- TypeScript
- Tailwind CSS
- Vite
- Firebase Authentication
- Firestore Database
- Cloudinary
- React Router
- React Hook Form

---

## 🔐 Autenticação

A autenticação foi implementada utilizando o **Firebase Authentication**.

Os usuários podem:

- Criar conta
- Fazer login
- Acessar funcionalidades privadas da aplicação

Rotas protegidas garantem que apenas usuários autenticados possam gerenciar anúncios.
